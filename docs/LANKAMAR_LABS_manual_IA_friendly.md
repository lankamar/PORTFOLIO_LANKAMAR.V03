## 1. Manifiesto estratégico y fundamentos
### Identidad institucional: sinergia híbrida
- Entidad puente entre práctica clínica tangible e innovación tecnológica.
- Rechazo explícito del hype/tendencias especulativas de IA; prioridad a evidencia y validación.
- Convergencia de clínica + docencia + competencia técnica (prompt engineering, RAG, LLM).

### Principio rector: Costly Signaling
- Diferenciación basada en profundidad verificable y evidencia empírica.
- No se “proclama” con adjetivos; se demuestra mediante artefactos.
- Portfolio como bitácora de investigación: trazabilidad, validación, honestidad intelectual.

### Metáfora visual: Escritorio de trabajo multidimensional
- Escenario central: escritorio monumental como taller de estudio, rigor académico y planificación.
- Anclajes constantes: materialidad madera; bibliografía UBA; carpetas Hospital de Clínicas; firma “Lankamar”.
- Variables estilísticas: elegir estilo visual según la naturaleza del tema (diorama/holografía/bloques/etc.).

## 3. Especificación técnica web (resumen)
- El sitio se concibe como “Registro de Investigación / Portafolio Anotado” personal, no comercial.
- Objetivo: costly signaling; atraer colaboraciones de alto nivel y filtrar demandas superficiales.
- UX: revelar evidencia con details-on-demand (panel lateral, expandibles) para densidad técnica sin saturar.
- Componente base sugerido: Artifact Card con Hipótesis, Evidencia (demo/código/logs), Limitaciones, Madurez (v0.1/v1.0/TRL).
